Project Name

Project Description

My Awesomecart is an innovative e-commerce platform designed to provide a seamless shopping experience. As a solo developer, I built this application to showcase my skills in full-stack web development.

Technologies & Tools Used

Python, Django, HTML, CSS,JavaScript, Bootstrap, jQuery, Git, GitHub

Features

  • User-friendly interface with intuitive navigation.
  • Secure user authentication and authorization.
  • Real-time product search and filter capabilities.
  • Efficient cart and checkout process with multiple payment options.
  • Responsive design for optimal viewing on all devices.
